## Задача 1

Дается два числа, найдите НОД.

Наибольший общий делитель (НОД) – это число, которое делит без остатка два числа и делится само без остатка на любой другой делитель данных двух чисел.

## Задача 2

Дана последовательность чисел, опредилите является она арифмитической

```
1 2 3 4 5 6
Ответ: Да
```

```
2 4 6 8
Ответ: Да
```

```
9 7 3
Ответ: Нет
```
